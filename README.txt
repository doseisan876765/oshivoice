【推しボイスNFCデモ：使い方】

このフォルダは、スマホでNFCタグをタッチしてボイスを時間帯に応じて再生するサンプルです。

＜構成ファイル＞
- index.html … ブラウザで開くと時間に応じたボイスを再生
- voices/ohayou.mp3 … 朝のボイス（自分で追加）
- voices/konnichiwa.mp3 … 昼のボイス（自分で追加）
- voices/oyasumi.mp3 … 夜のボイス（自分で追加）

＜公開方法＞
1. GitHubアカウントを作成（https://github.com）
2. 新しいリポジトリを作成（例：oshivoice-demo）
3. このフォルダ内のファイルをすべてアップロード
4. リポジトリの「Settings > Pages」から "main" ブランチを選んで公開
5. 表示されたURLをNFCタグに書き込む（例：https://yourname.github.io/oshivoice-demo/）

スマホでタッチすると、現在時刻に応じてボイスが流れます！
